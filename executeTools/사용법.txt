1. ApplicationConfiguration.conf 파일을 엽니다.

2. 각 설정들에 대한 바른 값을 넣습니다.
  - path : 대상이 되는 디렉토리 경로 
    ※ 해당 디렉토리 또한 포함입니다.
       예를들어, ".\"일 경우, ".\"에 있는 모든 소스 파일도 포함입니다.
  - logPath : 로그 파일 저장 위치
    ※ 이 로그 파일에 총 라인수가 출력됩니다.
  - walkingDeep : 서브 디렉토리 탐색 포함 여부
    ※ true라면 서브 디렉토리의 소스 파일도 모두 포함합니다.
  - countingEmptyLine : 공백 라인 포함 여부
    ※ true라면 공백 라인도 전부 포합합니다.
  - extensions : 대상 확장자
    ※ 카운팅할 대상 소스 파일의 확장자입니다.
  - singleComments : 제외할 comment 부호
    ※ 아무것도 안넣을 경우, 모든 주석도 카운팅에 포함시킵니다.
    ※ 넣을 경우, 넣은 주석 부호는 카운팅에서 제외됩니다.
  - groupComments : 제외할 comment 부호
    ※ 아무것도 안넣을 경우, 모든 주석도 카운팅에 포함시킵니다.
    ※ 넣을 경우, 넣은 주석 부호는 카운팅에서 제외됩니다.

3. start.bat 을 실행시킵니다.

-- 끝 --


주의사항
 comment의 경우, 만약 comment가 있는 줄에 comment가 아닌 source가 들어있는 경우,
카운팅의 대상이 됩니다.
ex) int a = 10; // a 변수
   -> 카운팅 대상에 들어감
